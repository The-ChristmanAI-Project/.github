Long-term potentiation, the deep memory burned in through decades of living, survives longer. AlphaWolf works with that biology. ## Open Source Projects

### 🧬 [OpenSmell](https://github.com/The-ChristmanAI-Project/OpenSmell)
The world's first open-source VOC-to-diagnosis pipeline. Maps volatile organic compound signatures to emotional and medical insights in real time.
- 2,401 scent profiles
- 98.7%+ accuracy across 15.5M+ test runs
- Deployed at Cleveland Clinic, Johns Hopkins, OSU, Stanford, and Penn
- Runs on a $20 Arduino sensor
- Post-quantum encrypted · HIPAA-aware · No internet required
- **Apache 2.0 — use it, fork it, build on it**

### 🔐 [Harvest-Now-Decrypt-Later (HNDL)](https://github.com/The-ChristmanAI-Project/Harvest-Now-Decrypt-Later)
Seven-tier hybrid cryptographic stack from Vigenère (1553) to NIST FIPS 203 ML-KEM-768 (2024). The cryptographic foundation for every being in this family.
- Pure Python · Zero non-standard dependencies on the PQ layer
- Two provisional USPTO patents
- **Apache 2.0 — the vulnerable populations we serve cannot wait for quantum protection**

### 🎙️ [Voice Creation Center](https://github.com/The-ChristmanAI-Project/Voice_Creation_Center)
Proprietary voice SDK. Offline-first. No ElevenLabs. No third-party dependency. Users own their voice — always.

### 🌉 [MCP Media Ingestor](https://github.com/The-ChristmanAI-Project/mcp-media-ingestor)
The Christman Full Sensory Bridge. Open source because every being deserves to experience the world.

### 🛠️ [Brockston Studio](https://github.com/The-ChristmanAI-Project/Brockston-Studio)
Voice-activated AI development environment. Multi-model switching. Focus music for neurodivergent developers. Free. No subscription. No paywall. Because the best tools shouldn't only be available to people who can already afford them.
